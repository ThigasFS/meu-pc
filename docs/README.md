# Plataforma Web para Montagem de Computadores Desktop

## Descrição
Sistema web que auxilia usuários na montagem de computadores personalizados,
verificando compatibilidade de componentes e sugerindo configurações.

## Tecnologias
- Node.js
- React
- MySQL
- Web Scraping
